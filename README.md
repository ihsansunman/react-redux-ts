# react-redux-ts

## Run Locally

Clone the project

```bash
  git clone https://github.com/ihsansunman/react-redux-ts
```

Go to the project directory

```bash
  cd react-redux-ts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
